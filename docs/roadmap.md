# Roadmap

## Phase 1 - Foundation MVP
- [x] Project and user management, JWT auth
- [x] GitHub Actions, TestRail, Jira connectors
- [x] JUnit XML and TestNG XML report upload
- [x] Execution pass rate and flaky test detection
- [x] Coverage heatmap and automation ratio
- [x] KPI dashboard with 9 metrics
- [x] Demo mode with artificial data
- [x] WCAG AA accessible dark/light theme
- [x] 281 unit/integration tests

## Phase 2 - Analytics Depth
- [x] Full defect analytics with MTTD/MTTR computation
- [x] Test history side-drawer (per-test execution trend)
- [x] Re-run tracking (original vs masked failure rate)
- [x] WebSocket live run streaming
- [x] Jira Stories connector with Epic sync and story-to-test-case linkage
- [x] Epic Coverage drill-down (Epic → Stories → Test Cases)
- [x] Flaky test inline drill-down (test history drawer on Runs page)
- [x] Connector export/import (JSON backup and cross-project transfer)
- [x] Docker Compose full-stack deployment with health checks (PostgreSQL, Redis, API, Web)
- [x] 550+ unit/integration tests
